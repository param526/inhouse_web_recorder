How this panel works

This panel orchestrates your Selenium session, the recorder script running in the browser, and the Java backend that generates Gherkin features.

  • Launch Browser: Opens a Selenium-driven browser instance at the URL you provide.
  • Recorder Script: Injected into the page, it records navigations, clicks and input changes.
  • Save & Quit: Persists all recorded actions into a JSON file and shuts down the browser.
  • Generate Gherkin: Sends your Feature & Scenario names to the Java app, which turns the recorded actions into a .feature file.
